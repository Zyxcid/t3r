# T3R
  
A minimalist, portable 20-20-20 rule timer for Windows. 

The 20-20-20 Rule: Every 20 minutes, look at something 20 feet (6 meters) away for 20 seconds to reduce digital eye strain.

Features: System Tray Integration, Break Overlay, Auto-Start.
